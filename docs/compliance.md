# 合规与风控清单（必须遵循）

## 总原则
1. 只使用**官方开放 API / SDK / 官方组件**；不得使用第三方/逆向接口。
2. 所有 **appid/appkey/签名密钥/ticket 换取** 等敏感逻辑必须在 **BFF（服务端）** 完成，客户端只拿短期结果。
3. 对外部平台请求必须做：**缓存、限流、重试退避、错误映射、审计日志**。

## QQ 音乐（OpenAPI）
- 需要签名校验与频控控制；默认频控较严格（公开文档提到 500 次/分钟）。
- 文档提示：非中国大陆 IP 可能被拒绝访问；部署地域需要符合要求。
- 文档提示：生产环境压测可能导致降频控/下线 appid（禁止压测）。
- 可能存在听歌流水上报/验收要求：尽早确认你的应用类型与合规清单。

参考： https://developer.y.qq.com/docs/openapi

## 酷狗（mini 酷狗 H5 组件）
- ticket 需通过服务端网关获取，有效期 2 小时，必须缓存并定期续期，避免频繁换票触发风控。
- H5 侧通过官方脚本初始化；客户端只能承载与控制，不应在客户端保存 appkey。

参考：  
- H5 接入 https://open.kugou.com/docs/mini-player/#/h5?v=3  
- ticket 获取 https://open.kugou.com/docs/mini-player/#/ticket

## 网易云 / 番茄畅听
- 未获得明确官方可接入 API/SDK 的情况下，只能 external 跳转，不做“统一播放”。
